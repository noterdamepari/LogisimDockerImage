# LogisimDockerImage
---
#### Перед выполнением инструкции убедитесь, что у вас на пк установлен Docker!!
1. Сборка образа:
	Перейдите в папку с клонированным репозиторием и выполните данную команду
	`docker build -t logisim-docker .`
2. Запуск контейнера:
	`docker run -d --name Logisim -p 5800:5800 -p 5900:5900 logisim-docker`
3. Доступ осуществляется через любой браузер установленный у вас на пк по адресу **127.0.0.1:5800**
4. Файлы прокидываются туда и обратно через команду:
	`docker cp`
	Подробнее как она работает можете ознакомиться в документации docker https://docs.docker.com/reference/cli/docker/container/cp/
	Не поддерживает копирование и вставку текста, из-за чего рекомендую не использовать сложные названия для файлов!
Основано на образе: https://github.com/jlesage/docker-baseimage-gui
